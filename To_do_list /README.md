# To-Do List Application

## Project Overview

The To-Do List Application is a Python-based project that helps users manage their daily tasks efficiently. It allows users to add, view, and remove tasks through a simple command-line interface, making task management easy and organized.

## Objective

The objective of this project is to develop a simple task management system that enables users to keep track of their activities and improve productivity.

## Features

- Add new tasks
- View all tasks
- Remove completed tasks
- Menu-driven user interface
- Easy task management

## Technologies Used

- Python

## Project Workflow

1. Display the main menu options.
2. Accept user input for different operations.
3. Add tasks to the task list.
4. Display all available tasks.
5. Remove selected tasks from the list.
6. Continue operations until the user chooses to exit.

## Results

The application successfully manages tasks by allowing users to add, view, and delete tasks through a simple and interactive interface.

## Learning Outcomes

- Understanding Python fundamentals
- Working with lists and data storage
- Implementing loops and conditional statements
- Handling user input effectively
- Building menu-driven applications
